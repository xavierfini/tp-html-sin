Description de mon site

Mon site présente le principe de base d'un système qui est géré par une carte programmée. Le but est de faire comprendre comment un système de ce type fonctionne.
Pour aider à la compréhension on utilise un exemple simple:L'arroseur automatique

Description des pages

La première page explique le principe de fonctionnement de l'arroseur
La page 2 explique le principe des entrées et des sorties pour une carte électronique
La page 3 donne le prnicipe du fonctionnement de la carte ARDUINO et de sa programmation
